## User

quiero diseñar la interfaz para una aplicación de escritorio, voy a usar tauri rust. es una aplicación para controlar el volumen de manera spatial. entonces por ejemplo, cargo doss aplicaciones que emiten audio, ambos por el mismo output device, el default, de hecho podría considerar que es el único disponible, en la práctica son mis auriculares bluetooth.

por default las aplicaciones están a la misma distancia del centro, y cargan con volumen default. deseo que esta aplicación me permita acercarme más a una aplicación al mismo tiempo que me alejo de otra, de manera que es más natural la piiorización del volumen, para empezar solo requiero controlar 2 aplicaciones simultáneamente.

ojo, solo es para el output de audio, es decir no es para controla micrófono, y el movimiento solo será en el eje horizontal, más a la derecha o más a la izquierda, movimiento que esspero poder controlar con las flechas del teclado, y con drag de mouse.
